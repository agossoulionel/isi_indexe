# Étude sur l'insomnie — Étudiants en sciences de la santé (Bénin)

Questionnaire bilingue FR/EN combinant données sociodémographiques, facteurs académiques/mode de vie, et l'Index de Sévérité de l'Insomnie (ISI).

## Accès au questionnaire

https://agossoulionel.github.io/isi_indexe/

## Description

Outil numérique conçu pour une étude de prévalence et de facteurs associés à l'insomnie chez les étudiants en sciences de la santé. L'échelle ISI (Morin, C., 1993) est utilisée pour évaluer la sévérité de l'insomnie. Les réponses sont collectées automatiquement dans une feuille Google Sheets.
